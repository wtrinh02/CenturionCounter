The Code for Centurion Counter


                        //get the values from the page
                        //starts or controller function
                        function getValues(){
                            //get values from  the page
                            let startValue = document.getElementById("startValue").value;
                            let endValue = document.getElementById("endValue").value;
                            //parse into integers
                            startValue= parseInt(startValue)
                            endValue = parseInt(endValue)
                        
                        
                            if(Number.isInteger(startValue) && Number.isInteger(endValue)){
                                 
                                if(startValue > endValue){
                                    alert("Start Value has to be less than or equal to end value");
                                    return;
                                }
                        
                                let numbers = generateNumbers(startValue, endValue)    
                                 displayNumbers(numbers)
                            }
                            else{
                                alert("You must enter integers!")
                            }
                        
                        
                        }
                        //generate numbers from startValue to endValue
                        //logic function(s)
                        function generateNumbers(startValue, endValue){
                            let numbers = [];
                            for (let i = startValue; i <= endValue; i++) {
                                numbers.push(i)
                            }
                            return numbers;
                        }
                        
                        //display the numbers and mark even numbers bold
                        //display or view functions
                        function displayNumbers(numbers){
                        
                            let templateRows="";
                            let className=";"
                        
                            for(let i = 0; i < numbers.length; i++){
                                let number = numbers[i];
                                if(number % 2 == 0){
                                   className = "even";
                                }
                                else{
                                    className = "odd";
                                }
                                //This doesn't render correctly with Prism see the Source code
                                templateRows+= `${number}`;
                            }
                            document.getElementById("results").innerHTML = templateRows;
                        }

The Code is structured in three functions.

getValues

getValues is the controller function of the coding challenge. The function gets the start and end values from the input of the page using the getElementById function to access the value of the input. getValues then parses the inputs into a number and checks if the inputs are valid. If so, it calls generateNumbers to create an array of numbers and uses the array to call displayNumbers to display the numbers in a table. Else an alert will pop up for an invalid input.

generateNumbers

generateNumbers is a function that takes the parameters of startValue and endValue. The function loops from startValue to endValue and adds all the numbers including and in between the two values into the numbers array.

displayNumbers

displayNumbers is a function that takes in an array of numbers. The function loops through the array and checks if the current number is even or odd by using the modulo(%) operator. If the number is even, the function sets the string className equal to even else className would be set to odd. After, during the loop, the function would add html elements to templateRows using backticks(`) to create table rows and table data with class even or odd with the number as data.
